# Simple Calculator

This is a simple calculator program written in C. It performs basic arithmetic operations:
- Addition
- Subtraction
- Multiplication
- Division

## How to Run
1. Compile the code:
   ```bash
   gcc calculator.c -o calculator

2. Run the program:

./calculator
